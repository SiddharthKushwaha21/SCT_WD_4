# 📝 Task 04 – Advanced To-Do Web Application

Web Development Internship – SkillCraft Technology

---

## 📌 Project Overview

This project was developed as part of **Task 04 of the Web Development Internship at SkillCraft Technology**.

It is a modern task management web application that allows users to create, manage, and organize daily tasks efficiently. The application includes task creation, date and time scheduling, task completion tracking, filtering functionality, and persistent data storage using LocalStorage.

The project focuses on building a clean, interactive, and responsive productivity tool with a modern glassmorphism UI design.

## ✨ Features

• Add new tasks  
• Assign date and time to tasks  
• Mark tasks as completed  
• Delete tasks  
• Filter tasks (All / Active / Completed)  
• Dynamic task rendering  
• LocalStorage data persistence  
• Smooth animations  
• Responsive design  
• Clean modern UI with glassmorphism effect
